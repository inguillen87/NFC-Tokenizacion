# nexID — Mini Brand Kit (v1)

## Palette
- Navy: `#0F172A`
- Navy Deep: `#0B1220`
- Slate (text): `#E2E8F0`
- Teal (accent): `#2FE1C3`
- Blue (support): `#60A5FA`

## Usage
- Default icon: `nexid-mark.svg`
- “Pulse” icon (hero / animations): `nexid-mark-pulse.svg`
- Wordmark: `nexid-wordmark.svg`
- Horizontal lockup: `nexid-lockup-horizontal.svg`
- Favicon: `nexid-favicon.svg`

## Clear space
Keep at least **12px** padding (at 160px icon size) around the mark.

## Min size
- Icon alone: 24px min
- Lockup: 220px min width

## Notes
- The dot + accent represents **scan/authentication**.
- The pulse variant suggests **NFC interaction** without cluttering the small icon.
